PrismQ Module Template
======================

Vítejte v dokumentaci PrismQ Module Template. Tato šablona poskytuje
standardizovanou strukturu pro vývoj PrismQ modulů v ekosystému
AI-powered generování obsahu.

   :maxdepth: 2
   :caption: Obsah:

   api

Funkce
------

* Konfigurace podle PEP standardů (PEP 517/518/621/660)
* Kontrola typů pomocí MyPy (PEP 484/526/544/561)
* Kvalita kódu s Ruff (PEP 8/257)
* Komplexní systém logování
* Nastavení testovacího frameworku (pytest)
* Vývojové skripty pro Windows

Cílová platforma
----------------

* **Operační systém**: Windows
* **GPU**: NVIDIA RTX 5090 (32GB VRAM)
* **CPU**: AMD Ryzen procesor
* **RAM**: 64GB DDR5

Rychlý start
------------

1. Naklonujte repository template
2. Spusťte ``scripts\setup.bat`` pro instalaci závislostí
3. Nakonfigurujte prostředí pomocí ``.env`` souboru
4. Začněte vyvíjet váš modul
