PrismQ Module Template Documentation
=====================================

Welcome to the PrismQ Module Template documentation. This template provides a
standardized structure for developing PrismQ modules that are part of the
AI-powered content generation ecosystem.

.. toctree::
   :maxdepth: 2
   :caption: Contents:

   modules

Features
--------

* PEP-compliant configuration (PEP 517/518/621/660)
* Type checking with MyPy (PEP 484/526/544/561)
* Code quality with Ruff (PEP 8/257)
* Comprehensive logging system
* Test framework setup (pytest)
* Development scripts for Windows

Target Platform
---------------

* **Operating System**: Windows
* **GPU**: NVIDIA RTX 5090 (32GB VRAM)
* **CPU**: AMD Ryzen processor
* **RAM**: 64GB DDR5

Quick Start
-----------

1. Clone the template repository
2. Run ``scripts\setup.bat`` to install dependencies
3. Configure your environment with ``.env`` file
4. Start developing your module

For detailed information, see the README.md in the repository root.
